## 性能对比

```
┌────────────────┬──────────┬────────────┬─────────────┐
│     场景       │  首次    │ 缓存命中   │  后台刷新   │
├────────────────┼──────────┼────────────┼─────────────┤
│ 串行扫描       │  10.6s   │  10.6s     │    N/A      │
│ 并行扫描       │   5.0s   │   5.0s     │    N/A      │
│ 智能缓存       │  10.6s   │  14ms      │  14ms+200ms │
└────────────────┴──────────┴────────────┴─────────────┘
```

## 关键特性

1. **并行扫描**: 所有 Agent 同时工作
2. **快速返回**: 缓存立即可用 (14ms)
3. **智能刷新**: 后台检测变更，增量更新
4. **数据一致**: 列表用缓存，详情实时读取

## 使用方法

```bash
# 默认模式（智能缓存）
codesesh

# 禁用缓存
codesesh --no-cache

# 清除缓存
codesesh --clear-cache

# 性能追踪
codesesh --trace
```
